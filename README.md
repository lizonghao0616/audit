# 心智审计：胜算重塑计划

> 一个基于身体感知的神经可塑性训练工具 —— 用物理动作重塑心智模式，而非单纯的理论说教。

## 概述

本项目是一个**单页 Web 应用**，通过 10 道结构化题目对用户的心智状态进行扫描诊断，并根据得分提供针对性的**物理补丁**和**进阶指南**，帮助用户从生理层面重建认知弹性与决策稳定性。

## 核心机制

- **10 维度扫描**：围绕不确定性反应、专注力、情绪调节、知行合一等维度进行量化评估
- **物理补丁导向**：不依赖"讲道理"，而是给出可执行的呼吸训练、神经调节音频、饮食方案等物理动作
- **飞书同步**：完成诊断后，自动通过 Webhook 将结果推送至飞书机器人，用于后续跟进

## 技术栈

- 纯前端单页应用（无框架，无构建）
- [Tailwind CSS](https://cdn.tailwindcss.com) — 原子化样式
- [canvas-confetti](https://github.com/catdad/canvas-confetti) — 高分彩带动画
- **localStorage** — 手机号脱敏缓存
- **飞书 Webhook** — 诊断结果推送
